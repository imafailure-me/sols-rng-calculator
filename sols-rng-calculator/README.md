# Sol's RNG Aura Calculator

A community calculator for estimating Sol's RNG aura drop chances, average rolls, average time, drops per hour, and probability milestones.

Created by **WhiteYN**.

## Features

- Searchable aura list
- Biome/environment modes:
  - Normal gameplay (AFK)
  - Ignore biome effects
  - Currently in a specific biome/time/location
- Native biome rarity support
- Biome-exclusive aura handling
- Glitched and Dreamspace AFK factors
- Left-handed device presets
- Custom rarity mode
- Rarer-aura blocking toggle
- Probability milestones: 10%, 25%, 50%, 75%, 90%, 95%, 99%

## Running locally

Open `index.html` in a browser.

No build step is required.

## GitHub Pages

This project is ready for GitHub Pages.

Recommended setup:

1. Push the repository to GitHub.
2. Go to **Settings → Pages**.
3. Source: **Deploy from a branch**.
4. Branch: `main`.
5. Folder: `/ (root)`.
6. Save.

## Notes

Sol's RNG changes frequently. Update aura rarities, biome values, and device mechanics when the game changes.
