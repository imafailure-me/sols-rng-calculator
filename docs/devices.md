# Left-handed devices

The calculator includes non-event left-handed device presets and custom settings.

## Included presets

- Nothing
- Jackpot
- Flesh
- Gravitational
- Darkshader
- Pole Light Core Device
- The Thing
- Unfathomable Ruins
- Custom / manual

Event or limited devices were intentionally removed from the public UI.

## Pole Light Core Device

Pole Light Core uses the supplied formula from the calculator development notes.

The current implementation treats:

```text
m1 = 1
```

This corresponds to:

```text
VIP = none
Dorcelessness = no
```

The user is expected to enter their final in-game luck directly.

## Custom device settings

Custom mode lets users test:

- bonus roll interval
- bonus roll multiplier
- flat luck add
- luck multiplier
- timed buff multiplier
- timed buff uptime
- bonus-roll removal
